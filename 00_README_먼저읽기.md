## ⚠️ 가장 먼저 — 이 자료의 신뢰도와 한계

이 리서치는 **여러 AI 에이전트가 실제 웹(유치원알리미 정보공시, 학교/유치원 공식 홈페이지, 네이버 블로그, 육아 정보 집계사이트 등)을 검색해 수집한 사실**만 정리한 것입니다. **임의로 지어낸 정보는 없습니다.** 확인되지 않은 항목은 모두 **"미확인"**으로 표기했습니다.

**다만 아래 한계가 있으니 반드시 감안하세요:**

1. **검색 도구가 미국 기준(US)이라** 네이버 블로그·맘카페 본문에 직접 접근이 자주 차단되었습니다. 그래서 **후기(특히 부정평가) 표본이 전반적으로 적습니다.** "부정평가 확인 안 됨"은 **"단점이 없다"가 아니라 "공개적으로 확인하지 못했다"**는 뜻입니다.
2. **유치원알리미(공식 원비 공시)** 가 자바스크립트 기반이라 일부 기관은 원비를 못 읽었습니다. **공식 원비가 확인된 곳은 한양여대부속·무학초병설 2곳뿐**이며, 미타·에덴은 블로그/검색 기반(재확인 필요), 나머지 사립은 원비 미확인입니다.
3. **셔틀 운행 권역은 매년·주소별로 달라집니다.** "통학버스 운영" 사실이 확인돼도 **금호/옥수 정차 여부는 대부분 미확인**이니, 입학 연도 기준으로 반드시 원에 직접 확인하세요.
4. **실재가 확인되지 않은 기관**도 있습니다(예: '서울옥정초병설유치원/옥수동' = 검색 0건, 명칭 오기 추정 / '율화유치원'·'서울응봉초병설' = 도구 차단으로 미검증). → `05_영어유치원_및_미검증기관.md` 참고.

> **결론적으로 이 자료는 "1차 후보 압축 + 전화 문의 체크리스트"로 활용**하시고, 최종 결정 전 **현장 방문 상담 + 유치원알리미 직접 조회 + 지역 맘카페 검색**을 권장합니다. 문의용 체크리스트는 `06_체크리스트_문의사항.md`에 정리했습니다.
