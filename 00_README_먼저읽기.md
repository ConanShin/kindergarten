# 성동구(금호·옥수 중심) 유치원 리서치 — 먼저 읽기

> **조사일:** 2026-06-21 (보강 업데이트 반영) · **대상 아동:** 2023년생(2027년 3월 **만3세반/연소반** 입학 예정)
> **거주·우선순위:** 금호동·옥수동 거주(셔틀 운영 시 원거리 허용) / **영어·비용·셔틀·방과후 4대 우선순위** / 국공립·사립·영어유치원 **전체 비교**

---

## 🚨 가장 중요한 발견 2가지 (보강 조사로 확인)

1. **경동유치원은 만3세반(2023년생 대상)을 운영하지 않습니다.** 2026학년도 6학급 전부 만4·5세, 공식 홈페이지도 6·7세반만 모집. → **2027년 만3세 입학 후보에서 사실상 제외**(만4세는 2028년 입학 가능). 그래서 1순위가 바뀌었습니다.
2. **여러 사립의 공식 원비(유치원알리미 2026 공시)를 확보**했고, **영어유치원 실데이터**도 찾았습니다(특히 **트리니엘어학원** = 금호동 소재, 셔틀이 금호·옥수까지 운행, 만3세부터, 종일 월 130~153만원).

---

## ⚠️ 이 자료의 신뢰도와 한계 (먼저 읽기)

이 리서치는 **여러 AI 에이전트가 실제 웹(유치원알리미 공식 공시, 학교/유치원 홈페이지, 네이버 블로그, 육아 정보 사이트 등)을 검색해 확인한 사실**만 정리했습니다. **임의로 지어낸 정보는 없으며**, 확인 못 한 항목은 **"미확인"**으로 표기했습니다.

- **후기(특히 부정평가) 표본이 적습니다.** 검색 도구가 네이버 블로그/맘카페 본문에 접근이 자주 막혀서입니다. "부정평가 확인 안 됨"은 **"단점이 없다"가 아니라 "공개적으로 확인 못 했다"**는 뜻입니다.
- **원비**: 유치원알리미 공시로 **만3세 학부모부담금이 확인된 곳**(미타·세성·옥수삼성·아이들세상·한양여대·무학초병설)과, **블로그/검색 기반(재확인 필요)**(에덴·옥수예일), **미확인**(경동·금호·성심)이 섞여 있습니다.
- ※ 공시 '학부모부담금'은 **누리과정 유아학비 지원이 반영된 학부모 실부담 기준**으로 통용되나, 지원금 적용·연도별 인상으로 실제 청구액과 차이날 수 있어 **원 확인 권장**.
- **셔틀 권역은 매년·주소별로 달라집니다.** "운영"이어도 **금호/옥수 정차 여부는 입학 연도 기준 직접 확인** 필수.
- **실재 미확인 기관**도 있습니다(옥정초병설=명칭 오기 추정, 율화·응봉초병설=도구 차단). → `05` 참고.

> **결론: 이 자료는 "1차 후보 압축 + 전화 문의 체크리스트"로 활용**하고, 최종 결정 전 **전화 문의 + 유치원알리미 직접 조회 + 현장 방문 + 맘카페 검색**을 병행하세요. 체크리스트는 `06`에 있습니다.

## 한눈에 보는 핵심 주의사항

- **경동유치원**: ❌ **만3세반 미모집** → 2027년 입학 불가(만4세 2028년만 가능)
- **한양여대부속**: 🚨 **종일반은 셔틀 미운영(반일반만 셔틀)** → 맞벌이로 종일반+셔틀 동시 필요 시 부적합. 방과후 특성화 분기 약 100만원 추가
- **에덴·아이들세상·옥수예일**: 만3세 원비가 높은 편(월 35만~43만원) → 가성비 중시 시 유의
- **국공립 병설(무학·옥수초·금호초)**: **셔틀 미운영/미확인** → '셔틀로 원거리 통원' 전제와 충돌. 도보·자가 통원 가능할 때만
- **미타·성심**: 셔틀 '운영 vs 미운영' 출처 상충 → 직접 확인
- **영어유치원(트리니엘·SLP 등)**: 월 100만원 이상 고비용 + 처음학교로 대상 아님(개별 접수)
- **입학 일정/경쟁률**: 2027학년도 설명회·접수 일정은 **2026년 가을 공지 예정**(현재 미공지). 유치원별 공식 경쟁률은 비공개 → 충원율로 갈음. 자세히는 `08` 참고
