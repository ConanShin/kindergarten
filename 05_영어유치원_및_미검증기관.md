# 영어유치원(유아 영어학원) + 실재 미검증 기관

## 먼저 — "영어유치원"의 성격

흔히 말하는 **'영어유치원'은 법적으로 유치원이 아니라 '유아 대상 영어학원(어학원 유치부)'** 입니다.
- **유치원알리미 공시 대상 아님** → 원비가 공개되지 않거나 학원비알리미(smileus)·민간 사이트로만 일부 확인.
- **처음학교로 대상 아님** → 개별 등록·상담·접수.
- 비용이 **월 100만원 이상**으로 일반 유치원보다 훨씬 높고, 급식·교재·특별활동비가 별도인 경우가 많습니다.

> 이번 보강 조사로 **트리니엘·SLP·JLS·CDI·폴리**의 실데이터를 확보했습니다(영어 심화 2건은 세션 한도로 실패). 영어를 최우선하신다면 아래를 출발점으로 **직접 상담·확인**하세요.

## ⭐ 1. 트리니엘어학원 — 금호·옥수 통원에 가장 적합한 영어유치원

- **위치**: 서울 성동구 독서당로51길 32, 영조빌딩 1~3층(금호동1가, **신금호역 인근**) · **☎ 02-2294-2143** · 공식 english-kindergarten.kr
- **실재**: 확인됨(edufindkorea 학원 등록 uid 162086, 사람인 '트리니엘성동본원' 기업정보 직원 15명, learns.academy)
- **대상**: **만 3세 ~ 초등**(영어유치부 + 초등부). 만3세부터 입학 가능
- **셔틀 ★**: 픽업 운영 — 권역 **금호동·옥수동·행당동·왕십리**(공식·learns.academy 기준). → **금호·옥수 거주 통원에 가장 현실적인 영어유치원** (성수/응봉/마장은 이번엔 미확인)
- **비용**: 종일반(주5회, 6시간) **월 약 130~153만원**(2025년 learns.academy 등록 교습비, 공식 표기와 일치). 입학금·교재비 별도(미확인), **2026년 원비 미확인**
- **커리큘럼**: 하브루타 기반 토론·질문 중심 '영재 언어교육' 표방. Reading, 하브루타 북클럽, Think Wide, Phonics, Language Art, Math, TOEFL 등. 소수정원
- **원어민(중요 한계)**: 공식은 '원어민 + 이중언어 교사 공동수업'이라고만 표기 → **원어민 교사 수·일과 중 원어민 수업 비율 미확인.** 정통 영어몰입(원어민 단독 종일)보다 **한국인 교사 비중이 큰 하이브리드일 가능성** → 상담 시 직접 확인 권장
- **평판**: learns.academy 4.7/5(3건, 표본 적음) — '아이가 좋아함', 소수 지도, 시설 우수. 독립 후기 미확인
- ⚠️ '아이비리그 연구진 공동개발' 등은 자체 마케팅 표현(제3자 검증 불가). 교습비는 2025년 기준 → 2026 인상 여부 확인 필요
- **꼭 물어볼 것**: 2026 원비·입학금·교재비, **원어민 교사 수·원어민 수업 비율**, 학급당 정원, 옥수동 셔틀 정류장·소요시간

## 2. 성동에스엘피(SLP) 영어학당 — 유아 종일과정 보유

- **위치**: 서울 중구 왕십리로 407, 신당 파인힐 하나유보라 상가 B1-17(2층)(신당동, **약수·왕십리권**) — 명칭은 '성동'이나 행정구역은 중구
- **대상/과정**: 유아영어 A~D + 초등영어 A~C. **유아영어A = 종일제(영어유치원 풀데이)**. 정원 10~12명
- **비용**(2023.04.12 고시, **2026 인상 가능·최신 미확인**): 유아영어A(종일) **월 105만원**, B 45만원, C 30만원, D 21만원
- **커리큘럼**: 서강대 SLP 프랜차이즈, 유아·초등 영어 전문, 강사 11명
- **셔틀**: 운영 여부 명시 미확인. 옥수동에서 차량 약 10분, 금호동도 접근권
- **평판**: 신뢰 후기 미확인(prompie 0건, 강남엄마 접근 차단)
- **출처**: 학원비알리미(smileus), youngu.kr(영유랭킹), prompie

## 3. 폴리(POLY)어학원 성동캠퍼스

- **위치**: **서울 중구 다산로 162**(신당동, 약수빌딩 1~2층, **약수역 인근**) — '성동'은 캠퍼스명, 실제 중구
- **대상**: 3~7세 · 정원 약 235명
- **비용**: 비공개(we-kidspot '월 교습비 7만원 이상', 영유랭킹 '분당 191원, 급식·교재·특활 별도') → 정확 금액 문의
- **커리큘럼**: 프랜차이즈 체계, 리딩·말하기·작문 중심, 원어민 비중 높음, e-POLY·영어도서관
- **셔틀**: 미확인
- ⚠️ **부정평가(주로 초등부/브랜드 일반)**: 시험(먼슬리·레벨테스트) 스트레스, 담임제 미흡, 문법 별도, 난이도 높음, 수업 늦게 끝남, 주차 협소. **유치부 전용 후기는 거의 없음**
- **출처**: poly-english.com, prompie(4.0/5), sjhlove.tistory, we-kidspot, youngu.kr

## 4. 정상어학원(JLS) 성동분원 / 청담어학원(CDI) 성동 — 유아 종일은 불확실

- **정상JLS 성동분원**: 성동구 금호동1가 행당로6길 2(금호벽산아파트 복지상가 3층). 2010년 개설. **유아 등록과목·유아활동영어 단서는 있으나 종일 킨더 운영 미확인**(방과후·오후 학원 형태 추정). 셔틀(픽업) 운영. 초등 원비 33~39만원/월. learns 4/5. **금호 도보권·옥수 차량 5~10분**
- **청담CDI 성동 브랜치**: 성동구 행당로 35(금북빌딩 3층). 크레버스 운영. **사실상 초·중등 영어 전문 — 미취학 유아 종일과정 미확인/없을 가능성.** 수강료 비공개(로그인). 셔틀 안내 페이지 존재
- → 두 곳 모두 **만3세 영어유치원(종일)으로는 부적합 가능성**. 유아 종일을 원하면 트리니엘·SLP가 현실적
